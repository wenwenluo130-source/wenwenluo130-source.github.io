Project Introduction

I designed Cover Letter Studio, an AI-powered workflow that helps candidates quickly generate tailored cover letters. By combining resume data, job descriptions, and optional context (like hiring manager information), the system identifies key motivation points and generates structured drafts. Users can adjust narrative style, tone, and experience level, then refine the draft with AI suggestions before exporting a polished, ATS-ready cover letter.
